# Customer Behavior Analysis

## Project Overview
This project analyzes customer purchasing behavior using Python and Exploratory Data Analysis (EDA). The analysis identifies revenue patterns, customer churn behavior, and purchasing trends through data visualization.

## Key Analysis Performed
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Revenue distribution analysis
- Customer churn analysis
- Monthly revenue trend analysis
- Correlation analysis

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
